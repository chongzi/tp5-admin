所有接口以post方式 接口域名：{php}echo config('http_type').$_SERVER['HTTP_HOST'];{/php} 常用接口地址说明

登入接口

  • 接口地址:{php}echo config('http_type').$_SERVER['HTTP_HOST'];{/php}/api/v1.platform/login

  • 请求参数

  
    {
      code: '',登入code
    }

  • 返回参数

  
   {
    code : 1和0(1:代表成功,0:代表失败)
    data:{
      openid:用户openid
    },
  msg:""(错误消息，失败才有)
}

应用列表接口地址

  • 接口地址:{php}echo config('http_type').$_SERVER['HTTP_HOST'];{/php}/api/v1.platform/platformList

  • 请求参数

  
    {
      app_id: 36  应用id
    }

  • 返回参数

  
   {
    code : 1和0(1:代表成功,0:代表失败)
    data:[{
      id:"",                应用id（用于点击）
      name:"",              应用昵称
      img:"",               应用图标
      url:""  ,             绑定的appid
      click_num:1          点击数
    }],
  msg:""(错误消息，失败才有)
}

应用点击接口地址

  • 接口地址:{php}echo config('http_type').$_SERVER['HTTP_HOST'];{/php}/api/v1.platform/appClick

  • 请求参数

  
    {
      id: 平台应用id
    }

  • 返回参数

  
   {
    code : 1和0(1:代表成功,0:代表失败)
    data:{
     
    },
    msg:""(错误消息，失败才有)
}

分享信息接口地址

  • 接口地址:{php}echo config('http_type').$_SERVER['HTTP_HOST'];{/php}/api/v1.platform/shareInfo

  • 请求参数

  
    {
      app_id: 36
    }

  • 返回参数

  
   {
    code : 1和0(1:代表成功,0:代表失败)
    data:[{
          ico:分享图片,
          title:分享标题
    }],
    msg:""(错误消息，失败才有)
}
